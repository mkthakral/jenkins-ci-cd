# jenkins-ci-cd

It is Jenkins CI/CD pipeline built for SAP Hybris Ecommerce project, but can be used for other applications.

## Tools
* Lighhouse
* ZAP
* JUnit
* Jacoco
* Sonar

## Stages
* Checkout Source Code
* Compile
* JUnit + CodeCoverage
* Sonar
* Build Deployment Package
* Deploy
* Test: LightHouse, ZAP, Integration
